Normal Distribution

Parameters: Mean (μ), Standard Deviation (σ)

PDF: f(x) = (1 / (σ√(2π))) * exp(-((x - μ)² / (2σ²)))

CDF: F(x) = 0.5 * [1 + erf((x - μ) / (σ√2))]

Mean: μ

Variance: σ²

Binomial Distribution

Parameters: Number of Trials (n), Probability of Success (p)

PMF: P(X = k) = C(n, k) * p^k * (1 - p)^(n - k)

CDF: F(k) = ∑ P(X = i) for i from 0 to k

Mean: np

Variance: np(1 - p)

Solver

Poisson Distribution

Parameters: Rate (λ)

PMF: P(X = k) = (λ^k * e^(-λ)) / k!

CDF: F(k) = ∑ P(X = i) for i from 0 to k

Mean: λ

Variance: λ

Solver

Exponential Distribution

Parameters: Rate (λ)

PDF: f(x) = λ * e^(-λx) for x ≥ 0

CDF: F(x) = 1 - e^(-λx)

Mean: 1/λ

Variance: 1/λ²

Solver

Uniform Distribution

Parameters: Lower bound (a), Upper bound (b)

PDF: f(x) = 1 / (b - a) for a ≤ x ≤ b

CDF: F(x) = (x - a) / (b - a) for a ≤ x ≤ b

Mean: (a + b)/2

Variance: (b - a)² / 12

Solver

Bernoulli Distribution

Parameters: Probability of Success (p)

PMF: P(X = k) = p if k = 1, 1 - p if k = 0

CDF: F(k) = 0 if k < 0, 1 - p if 0 ≤ k < 1, 1 if k ≥ 1

Mean: p

Variance: p(1 - p)

Solver

Gamma Distribution

Parameters: Shape parameter (k), Rate parameter (θ)

PDF: f(x) = (x^(k - 1) * e^(-x / θ)) / (θ^k * Γ(k)) for x ≥ 0

CDF: F(x) = ∫ f(t) dt from 0 to x

Mean: kθ

Variance: kθ²

Solver

Beta Distribution

Parameters: Shape parameters (α, β)

PDF: f(x) = (x^(α - 1) * (1 - x)^(β - 1)) / B(α, β)

CDF: F(x) = ∫ f(t) dt from 0 to x

Mean: α / (α + β)

Variance: (αβ) / [(α + β)²(α + β + 1)]

Solver

Negative Binomial Distribution

Parameters: Number of successes (r), Probability of success (p)

PMF: P(X = k) = C(k + r - 1, k) * (1 - p)^k * p^r

CDF: F(k) = ∑ P(X = i) for i from 0 to k

Mean: r * (1 - p) / p

Variance: r * (1 - p) / p²
